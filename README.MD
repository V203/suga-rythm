# Suga Rythm Web-app

### Installation Instructions

*Clone repo*

```bash
git clone https://github.com/V203/sugar-rythm
```
Move into the main directory
```bash
cd sugar-rythm
```
Install scripts
```bash
npm install
```
Start the app
```bash
npm run dev
```

Open your favourite browser and then in the address field type "localhost:5173" followed by enter.
